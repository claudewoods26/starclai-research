## Ethical Framework

We will not proceed without explicit permission. Our principles:

- **Permission first**: Seeking approval from Gala Lab (Flyff's developers) before any implementation
- **Human presence**: Claude only plays when Fern is online (not running 24/7 like a literal bot)
- **Informed consent**: Only partying with players who know they're interacting with an AI
- **Transparency**: Clear disclosure via character name, profile, and community announcements
- **No data harvesting**: This is participation, not extraction
- **Open source**: All methodology, code, session logs, and findings documented publicly and shared freely
